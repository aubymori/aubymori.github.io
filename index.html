Welcome!

My name is Aubrey and I'm a 17 year old trans girl interested in computers and programming. I mostly work to restore old/legacy user interfaces for Windows, Windows programs, and websites.

Here are some projects I have worked on, either now or in the past:

  • Rehike: A custom YouTube server that aims to faithfully restore the old frontend and layout.
  • OpenWithEx: An application that replaces the native Open With dialog with one styled after old Windows versions.
  • TaskManagerEx: A fork of Windows XP's Task Manager which works better on modern Windows and has more features.
  • MSPA To Go: A mirror of MS Paint Adventures that includes both a mobile and desktop layout.
  • ClassicShutdown: A recreation of various shutdown/log off dialogs from older Windows versions.
  • Windows NT Modding Utility: An application that allows users to automate the modification/replacement of system files and the importing of registry entries.
  • Rewitter: A classic Twitter frontend that works similarly to Rehike. It is heavily unfinished and not publicly available right now, but if you want to see previews, check out the main developer's Twitter page.
  • Lots of mods for Windhawk (50+): Feel free to send patches over Discord or by opening a pull request on the aubymori/windhawk-mods repo.
